Instructions

Quick Start

  1. Click Settings
  2. Choose the Competition
  3. Click Save
  4. Go back to the Main Screen, Click Scout
  5. Scout Matches

In-Depth Information

In order to use this app, the competition must be set. Doing so allows the software to keep data separate and to download match information.

After this point, internet access is not required, but strongly recommended.

At this point, matches can be scouted. If match information has been downloaded, then alliance information will be preloaded. However, if there is no internet access when the schedule is generated, this information cannot be downloaded and the alliance infomation must be inserted manually.

After matches have been scouted, the data can be consolidated onto one app (more info to come, still under development).

Using the Statistics menu, team members can view and edit collected data, view charts and graphs regarding the data, and view the best fit for alliance selections.

Notes Regarding Internet Access

Internet Access is required for the selection of the competition. This selection is then used to pull and preload match information (click the message after schedules generated), like alliances.

If Internet access is not available to download the schedule, then the scouter will have to manually enter the match information.
